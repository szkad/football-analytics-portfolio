# Football Analytics Portfolio

Portafolio de análisis de datos en fútbol usando Python, StatsBomb Open Data y herramientas modernas de visualización deportiva.

## Stack Técnico

- **Lenguajes:** Python, SQL
- **Procesamiento:** pandas, numpy, DuckDB
- **Visualización:** matplotlib, mplsoccer, seaborn
- **Modelado:** scikit-learn, socceraction
- **Datos:** StatsBomb Open Data

## Proyectos

### 1. Análisis de Eventos - StatsBomb Open Data
*En desarrollo*

Exploración de event data de competencias internacionales: anatomía de partidos, construcción de métricas avanzadas (xG, xT), visualizaciones tácticas y modelos predictivos.

## Estructura del Repositorio
notebooks/   Análisis exploratorios y experimentos
src/         Módulos reutilizables
data/        Datos crudos y procesados (no versionados)
outputs/     Visualizaciones y reportes generados
docs/        Documentación adicional

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```
